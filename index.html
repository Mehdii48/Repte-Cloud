Sitio web estático desplegado usando AWS Amplify Hosting con HTTPS.

Este sitio se sirve globalmente mediante el CDN integrado de AWS Amplify, mejorando el rendimiento, reduciendo la latencia y ofreciendo acceso seguro mediante HTTPS.

Arquitectura

Usuario → AWS Amplify (CDN integrado) → S3 gestionado por Amplify

Objetivos

  • Alojamiento estático gestionado por AWS Amplify
  • Caché global integrada
  • HTTPS automático gestionado por AWS
  • Despliegue continuo desde GitHub
