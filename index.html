Sitio web estático desplegado usando Amazon S3, CloudFront y HTTPS.

Este sitio se sirve globalmente mediante un CDN, mejorando el rendimiento, reduciendo la latencia y habilitando acceso seguro mediante HTTPS.

Arquitectura

Usuario → CloudFront (CDN + caché) → S3 (archivos estáticos)

Objetivos

  • Alojamiento estático en S3
  • Caché global con CloudFront
  • HTTPS con ACM
  • Control de caché e invalidaciones
